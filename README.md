Vienna
=====

Vienna is a utility library for WPF development.  It contains a number of different 
components that can be used within your WPF projects to simplify development.  These components
include converters, behaviors, markup extensions and utility helper classes.  
A brief list of these helpers follows:

* DragAndDropBehavior - a set of classes to simplify drag and drop operations in WPF
* ItemsControlBehavior - a behavior for stylizing alternate items inside an ItemsControl
* WindowFocusBehavior - a behavior for defining the control to receive focus when a window is loaded
* BetterBooleanToVisibilityConverter - a converter that adds many missing features over the built-in BooleanToVisibility converter
* BooleanToOpacityConverter - converts between booleans and opacities
* DebugConverter - helpful for validating & debugging runtime values
* NumberFormatConverter - Formats numeric values for display
* WordSeparatorConverter - Converts camel-cased strings into phrases
* ControlHelper - utilities for working with WPF Controls
* GlassHelper - utility for extending window glass into the client area of a window
* VisualTreeHelperEx - utilities for working with the visual tree
* ImageResourceExtension - Markup Extension for working with images from resource files
* StringResourceExtension - Markup Extension for working with strings from resource files
* DecimalValidationRule - ValidationRule for range-checking decimals
